# InfinityNikki-Scriptable-Widget

### 一个使用 [Scriptable](https://apps.apple.com/us/app/scriptable/id1405459188) 制作的无限暖暖小组件

当前支持：

- 昵称、等级、头像显示

- 日常、星海、周本的完成进度显示

- 体力、挖掘的剩余时间显示
